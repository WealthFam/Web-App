## 🚀 Overview

The WealthFam Frontend is a modern web application built for financial visualization and management. It leverages **Material Design 3 (M3)** principles to provide a premium, accessible, and highly responsive experience across all screen sizes.

> [!TIP]
> This is a decoupled micro-frontend. For full system orchestration, see the [Infra](../../Infra/) repository.

---

## 🛠️ Technical Stack

| Category | Technology |
| :--- | :--- |
| **Framework** | [Vue.js 3](https://vuejs.org/) (Composition API) |
| **Build Tool** | [Vite](https://vitejs.dev/) |
| **State** | [Pinia](https://pinia.vuejs.org/) |
| **Styling** | [Tailwind CSS](https://tailwindcss.com/) |
| **Analytics** | [Chart.js](https://www.chartjs.org/) |
